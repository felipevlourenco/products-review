## Project setup

```
yarn
```

### Lints and fixes files

```
yarn lint
```

### Format files

```
yarn format
```

### Test

```
yarn test
```

### Serve Client app and hot-reloads for development

```
yarn start:client
```

### Serve Server and hot-reloads for development

```
yarn start:server
```

### Run Server and client at the same time

```
yarn start
```
